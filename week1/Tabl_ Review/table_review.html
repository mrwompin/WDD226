Originally developed by Ramus Lerdor, PHP got it's start as set of scripting tools written in C known at the time as Personal Home Page Tools. These tools allowed user to create simple dynamic web applications such as guest books. PHP gradually began to be developed when the source code was released but it wasn't until 98 that PHP included object oriented programming support. As of today PHP, with it's related supporting projects such as PEAR, PECL, and documentation, is by some estimates installed on tens to hundreds of millions of domains around the world.
